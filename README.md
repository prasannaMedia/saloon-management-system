# Fabb6 - Salon & Spa Management Platform

A modern, feature-rich salon and spa management platform built with React, Tailwind CSS, and best practices. Similar to GlossGenius and Zenoti, this platform provides comprehensive booking, ecommerce, and business management features.

## üöÄ Features

### Customer-Facing Features
- **Smart Booking System**: Multi-step appointment booking with service, staff, and time selection
- **Services Showcase**: Comprehensive service catalog with categories, pricing, and packages
- **E-commerce Shop**: Full-featured product marketplace with filters, cart, and ratings
- **Responsive Design**: Beautiful UI that works on all devices

### Business Management Features
- **Admin Dashboard**: Real-time analytics and business insights
- **Revenue Tracking**: Monitor daily/monthly revenue and growth trends
- **Appointment Management**: View and manage all appointments
- **Staff Performance**: Track staff metrics and performance
- **Client Management**: Comprehensive client profiles and history

### UI/UX Features
- Gradient designs with pink/rose theme
- Smooth animations and transitions
- Modern card-based layouts
- Interactive components
- Mobile-responsive navigation
